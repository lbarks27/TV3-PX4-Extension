# TV3 Lander CAD References

Store lander CAD source files and exported render meshes here.

Active SIH runs do not consume visual meshes. Future visual adapters should use
the vehicle reference frame declared in `config/vehicles/tv3_lander_v1.yaml`:
origin at nozzle exit center, +X forward along the airframe, +Y vehicle right,
and +Z vehicle down.

Historical Gazebo-specific mesh naming guidance lives in
`deprecated/sim/gazebo/assets/cad/lander/README.md`.
